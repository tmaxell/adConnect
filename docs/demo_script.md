# AdConnect Copilot — демо-сценарий

Сценарий показа агента сборки кампаний, который **гарантированно не падает** и
выглядит аккуратно. Работает **без LLM-ключа** (детерминированный режим): подбор
аудитории, генерация креативов, прогноз и названия — на встроенной логике.

---

## 0. Подготовка (за 2 минуты до показа)

**Вариант A — Docker (рекомендуется для показа):**
```bash
cd adConnect
docker compose up --build        # дождаться, пока backend healthy
# открыть http://localhost:8080
```

**Вариант B — локально (два терминала):**
```bash
# терминал 1 — backend
cd adConnect/backend && source .venv/bin/activate
uvicorn app:app --port 8000
# терминал 2 — frontend
cd adConnect/frontend && npm run dev   # http://localhost:5173
```

Чек-лист перед показом:
- [ ] **Backend поднят первым.** Открыть `http://localhost:8000/api/health` → `{"status":"ok"}`. Если открыть UI раньше backend — в виджете мигнёт «Сервис временно недоступен» (не страшно, но некрасиво).
- [ ] Открыть UI, убедиться, что виден список «Advertising campaigns» (демо-кампании малого бизнеса) и логотип Eastwind.AdConnect.
- [ ] Окно браузера широкое (≥ 1280px) — тогда dock-режим виджета смотрится лучше всего.
- [ ] *(опционально)* чистый старт: остановить backend, удалить `backend/data/adconnect_agents.sqlite3`, поднять заново — список будет на демо-данных.
- [ ] *(опционально, для красивых названий/текстов)* положить `GOOGLE_API_KEY` в `backend/.env` — тогда имена и креативы генерирует LLM. Без ключа всё работает на шаблонах.

**Золотое правило показа:** ведите сборку **кнопками-чипами** в виджете (Канал → Подобрать аудиторию → Сгенерировать креативы → Выбрать → Отправить). Свободным текстом вводите только бюджет («Бюджет 30000») — он надёжно распознаётся.

## 1. Завязка (30 сек)

1. Показать экран **Ad Campaigns** — это продукт оператора: список кампаний малого бизнеса со статусами «Under moderation».
2. Нажать на **синюю иконку-искру** (FAB справа внизу) — открывается **AdConnect Copilot**. Сказать: «единое окно — собрать кампанию можно прямо в чате, не разбираясь в кабинетах».

---

## 2. Основной сценарий — SMS-кампания «вживую заполняет холст» (2–3 мин)

> Идея: чат ведёт, **холст продукта заполняется сам**, шаг за шагом.

1. В виджете нажать подсказку **«Создать рекламную кампанию»** (или ввести: **«Создай рекламную кампанию для фитнес-клуба»**).
   → На фоне холст переключается в **мастер создания кампании**, активен шаг **Sending Channel**. Слева — операторские каналы (SMS/Email), ниже — внешние сети (**Meta Ads** активна, Telegram/Google — «Скоро»).
2. В виджете нажать **«SMS»**.
   → Шаг-бар уезжает на **Segments**, заголовок становится «SMS Campaign».
3. Нажать **«Подобрать аудиторию за меня»**.
   → Агент предлагает 2–3 сегмента абонентской базы с охватом. Нажать, например, **«Выбрать: Active mobile users»** (или любой).
   → На холсте заполняются Geography/Interests/Age, справа обновляется **Audience reach** и **Price per message**.
4. Нажать **«Сгенерировать креативы»** → агент выдаёт 3 варианта текста. Нажать **«Вариант 1»**.
   → На холсте в шаге Message появляется выбранный текст.
5. Ввести в поле сообщение: **«Бюджет 30000»** и отправить.
   → Холст переходит на **Confirmation**: сводка аудитории, имя кампании, текст, охват и стоимость. Справа — посчитанная цена.
6. Нажать **«Отправить на модерацию»**.
   → Ответ агента: «Кампания … отправлена на модерацию. Списания и запуска не произошло». Холст показывает submitted-состояние.
   > Подчеркнуть **guardrail**: агент собирает и отправляет на модерацию, но **не запускает и не списывает деньги** без подтверждения.

7. Показать, что кампания **сохранилась**: в шапке виджета нажать **«+»** (новый диалог) → холст возвращается к списку **Ad Campaigns**, и новая кампания уже в списке (вверху).

---

## 3. Изюминка — Meta как канал (1–2 мин)

> Показывает, что один и тот же сценарий ведёт в внешнюю сеть с другой моделью оплаты.

1. Новый диалог (**«+»**). Нажать подсказку **«Создать рекламную кампанию»**.
2. На шаге канала нажать **«Meta Ads»**.
   → Заголовок «Meta Campaign». Сказать: «Meta — это не оплата за сообщение, а аукцион: аудитория оператора загружается как **Custom Audience** (по хешам телефонов), оплата за показы (CPM)».
3. **«Подобрать аудиторию за меня»** → выбрать сегмент.
   → Справа панель теперь показывает **Custom Audience**, **CPM** и **Ожидаемые показы** вместо цены за сообщение.
4. **«Сгенерировать креативы»** → выбрать вариант.
5. Ввести **«Бюджет 50000»**.
   → Холст: показы ≈ 166 666 при CPM 300 ₽; в сводке Confirmation — Placements (Facebook, Instagram, WhatsApp), Custom Audience, CPM, Est. impressions.
6. **«Отправить на модерацию»**. В списке кампаний канал отметится как **Meta**.

---

## 4. Эффектный финал (30 сек)

- В шапке виджета нажать **«развернуть»** (стрелки) → **dock-режим**: виджет занимает правую половину, холст продукта аккуратно ужимается влево, шапка остаётся на всю ширину. История диалогов — колонкой слева.
- Свернуть обратно. Сказать про **историю**: все диалоги и кампании хранятся и подгружаются (можно показать список диалогов по иконке-«гамбургеру»).

## Что НЕ делать на показе (чтобы ничего не полетело)

- ❌ Не открывать UI, пока backend не поднят (иначе мелькнёт «Сервис временно недоступен»).
- ❌ Не вводить аудиторию длинным свободным текстом — используйте чип **«Подобрать аудиторию за меня»** (детерминированный подбор по каталогу).
- ❌ Не жать **«Отправить на модерацию»** дважды — повторный submit идемпотентен (дубля не будет), но в демо это лишнее.
- ❌ Не перезагружать страницу в середине сборки ради «оживления» — виджет свернётся в иконку; холст сам восстановится из сохранённого черновика, но пауза смотрится хуже, чем плавный диалог.
- ✅ Бюджет вводите явно: «Бюджет 30000» / «Бюджет 50000».
- ✅ Если показываете без LLM-ключа — название кампании будет в родительном падеже («Фитнес-клуба»): это нормально для демо; с ключом названия и тексты получаются «человечнее».

## Резервный план

- Если backend «задумался» — кнопки-чипы идемпотентны, просто повторите клик.
- Если что-то совсем зависло — **«+»** (новый диалог) начинает с чистого холста; ранее собранные кампании остались в списке.
